# 6th Sem CGPA Calculator

A static CGPA planning website for sixth semester students.

## Deploy On Render

1. Push this folder to a GitHub, GitLab, or Bitbucket repository.
2. Open Render and create a new Static Site.
3. Connect the repository.
4. Use these settings:
   - Build Command: leave empty
   - Publish Directory: `.`
5. Deploy.

Render can also detect `render.yaml` from the repository root.
